6.1 伽罗瓦域（Galois Field）的存在法则

无限域（如 $\mathbb{R}, \mathbb{C}$）有无穷多种，但有限域展现出了极度的刚性：它们的规模只能是素数的幂次方，且同构意义下是唯一的。

关联知识点：素数的基石作用

存在性与唯一性定理

一个有限域的元素个数只能是 $q = p^m$（$p$ 是素数，$m$ 是正整数）。

对于任何给定的素数幂 $p^m$，恰好存在一个含有 $p^m$ 个元素的有限域（同构意义下唯一），记为 $GF(p^m)$ 或 $\mathbb{F}_{p^m}$。其中 $p$ 是域的特征（Characteristic）。

注意：当 $m=1$ 时，$GF(p)$ 即为我们熟悉的剩余类域 $\mathbb{Z}_p$。但当 $m > 1$ 时，$GF(p^m)$ 绝不等同于整数模 $p^m$ 的剩余类环 $\mathbb{Z}_{p^m}$（后者有零因子，不是域）。

6.2 扩域 $GF(p^m)$ 的多项式构造原理

要在大小为 $p$ 的基础上构造出大小为 $p^m$ 的域，我们必须借助多项式工具和"模运算"的思想。这与复数的构造（$\mathbb{R}[x]/(x^2+1)$）如出一辙。

关联知识点：不可约多项式的概念

利用不可约多项式构造

设 $P(x)$ 是 $GF(p)$ 上一个次数为 $m$ 的不可约多项式。

$GF(p^m)$ 的元素构成为 $GF(p)$ 上所有次数小于 $m$ 的多项式集合： $$a(x) = a_{m-1}x^{m-1} + \dots + a_1x + a_0 \quad (a_i \in GF(p))$$

由于每个系数有 $p$ 种选择，共有 $p^m$ 种不同的多项式。其运算规则为：

  1. 加减法：多项式系数按对应次序在 $GF(p)$ 中进行模 $p$ 的加减法。
  2. 乘法：两个多项式先进行普通乘法，然后将结果除以 $P(x)$ 取余式。由于 $P(x)$ 不可约，这保证了非零元素都有乘法逆元。

6.3 本原多项式与元素的表示形态

为了简化乘法运算，我们引入生成元（本原元）的概念，从而利用指数运算将乘法转化为加法。

本原多项式（Primitive Polynomial）

一个次数为 $m$ 的多项式 $P(x)$，如果它是不可约多项式，并且它的根 $x$ 是 $GF(p^m)$ 乘法群中的生成元，则称 $P(x)$ 为本原多项式。即 $x$ 的所有幂次能生成除了 $0$ 以外的所有 $p^m - 1$ 个元素。

在 $GF(p^m)$ 中，元素有四种等价的表示形态：

形态形式优势操作
多项式形式$a_{m-1}x^{m-1} + \dots + a_0$理论推导与代数运算
向量（系数）形式$(a_{m-1}, \dots, a_1, a_0)$加减法（按位模 $p$）、存储
整数（或二进制字）$\sum a_i p^i$（当 $p=2$ 则是十六进制）计算机中的数据表示与传输
指数形式$x^k$（$x$为生成元）乘法、除法（指数相加减模 $p^m-1$）

6.4 工业应用：AES中的 $GF(2^8)$ 运算体系

现代密码标杆 AES 算法的所有字节变换均在 $GF(2^8)$ 中完成。

关联知识点：多项式欧几里得算法与逆元求解

AES $GF(2^8)$ 参数规范

底层域：$GF(2)$，系数只有 $0, 1$，加法和减法均为异或（XOR）运算。

模多项式：不可约多项式 $m(x) = x^8 + x^4 + x^3 + x + 1$（十六进制表示为 $\text{0x11B}$）。

乘法操作 $x \cdot a(x)$ (xtimes)：多项式乘以 $x$，相当于比特向量左移1位。如果最高位溢出（移出 $x^8$），则由于 $x^8 \equiv x^4 + x^3 + x + 1 \pmod{m(x)}$，需将结果与 $\text{0x1B}$ 进行异或。

S-盒的核心：AES 的 S 盒主要由输入字节在 $GF(2^8)$ 中的乘法逆元与仿射变换叠加而成。求逆元是提供强非线性的密码学基石。

6.1 判断有限域的存在性 基础
判断以下规模的集合是否可能构成有限域：(1) 9 (2) 10 (3) 15 (4) 32 (5) 16。
考点伽罗瓦域存在定理：元素个数必为 $p^m$。
  1. (1) 9 = $3^2$，是素数幂，能构造 $GF(3^2)$。
  2. (2) 10 = $2 \times 5$，有两个不同的素因子，不能构成有限域。
  3. (3) 15 = $3 \times 5$，有两个素因子，不能构成有限域。
  4. (4) 32 = $2^5$，是素数幂，能构造 $GF(2^5)$。
  5. (5) 16 = $2^4$，是素数幂，能构造 $GF(2^4)$。
✅ 总结：在代数中，具有10个元素或15个元素的交换环确实存在（如剩余类环 $\mathbb{Z}_{10}$），但它们必然包含零因子，绝不可能构成一个没有零因子的完美"域"。
6.2 GF(p) 中的运算本质 基础
在 $GF(7)$ 中计算 $\frac{3 + 5}{4 \times 6}$。
考点$GF(p)$ 即 $\mathbb{Z}_p$，所有运算均为模 $p$ 的算术，除法即为乘逆元。
  1. 分子：$3 + 5 = 8 \equiv 1 \pmod 7$。
  2. 分母：$4 \times 6 = 24 = 3 \times 7 + 3 \equiv 3 \pmod 7$。
  3. 式子化简为求解在 $GF(7)$ 中的 $1/3$，即求 $3$ 的乘法逆元。
  4. 寻找 $x$ 使 $3x \equiv 1 \pmod 7$：由于 $3 \times 5 = 15 = 2 \times 7 + 1 \equiv 1$，故逆元为 $5$。
  5. 结论：计算结果为 $5$。
✅ 总结：对于一次幂素数域，没有多项式的概念干扰，所有算子直接回归第2章初等数论的模运算。
6.3 GF(2^m) 元素的二进制形式转换 基础
在 $GF(2^8)$ 中，将十六进制数 $\text{0x5A}$ 转换为多项式形式。
考点十六进制 $\to$ 二进制 $\to$ 降幂排列作为多项式系数。
  1. 十六进制分解：$\text{5} \to 0101$，$\text{A} \to 1010$。
  2. 拼接二进制向量：$01011010$。
  3. 从左至右，赋予从高到低的多项式基底 $x^7$ 到 $x^0$：
    $0x^7 + 1x^6 + 0x^5 + 1x^4 + 1x^3 + 0x^2 + 1x^1 + 0x^0$
  4. 结论：化简为 $x^6 + x^4 + x^3 + x$。
✅ 应用：在AES硬件实现或编程题中，数据全都是以十六进制存放的，第一步总是展开为多项式才能进行代数推演。
6.4 GF(2^m) 中的加法特性 基础
在 $GF(2^8)$ 中，计算十六进制数 $\text{0x5A}$ 与 $\text{0x3F}$ 的加法和减法结果。
考点在特征为2的域中，加法和减法完全等价，均表现为按位异或（XOR）。
  1. 将十六进制转为二进制：
    $\text{0x5A} = 0101 1010_2$
    $\text{0x3F} = 0011 1111_2$
  2. 执行按位异或运算（不进位）：
    $0101 1010$
    $\oplus$ $0011 1111$
    ----------------
    $0110 0101$
  3. 将结果转回十六进制：$0110 \to 6$，$0101 \to 5$。
  4. 由于特征为2，$-1 \equiv 1 \pmod 2$，减法就是加法，结果相同。
  5. 结论：$\text{0x5A} + \text{0x3F} = \text{0x65}$，减法同样为 $\text{0x65}$。
✅ 总结：在 $GF(2^m)$ 中永远不要使用带进位的算术加法！这是初学者最容易犯的低级错误。
6.5 GF(2^3) 模多项式乘法初步 中等
构造 $GF(2^3)$，不可约多项式为 $P(x) = x^3 + x + 1$。计算 $(x^2+1) \cdot (x^2+x)$ 的结果多项式。
考点两阶段操作：普通展开合并 $\to$ 长除法或降次代换求余数。
  1. 普通展开：$(x^2+1)(x^2+x) = x^4 + x^3 + x^2 + x$。
  2. 因为是 $GF(2)$，系数不变（无需异或消除同类项，全不同次）。
  3. 由于最高次 $4 \ge 3$，必须模 $P(x)$。
  4. 使用降次公式法：$P(x)=0 \Rightarrow x^3 \equiv x + 1$。
  5. 计算 $x^4$：$x^4 = x \cdot x^3 \equiv x(x+1) = x^2+x$。
  6. 代入原多项式：
    $(x^2+x) + (x+1) + x^2 + x$
    $= (x^2+x^2) + (x+x+x) + 1$
    $= 0 + x + 1$ （因为 $1+1+1=1$在 $GF(2)$ 中）
  7. 结论：结果为 $x + 1$。
✅ 总结：降次公式法（用低次替换高次）通常比硬写长除法更不容易算错符号，而且极具编程直觉。
6.6 AES xtimes 操作 (乘 $x$) 中等
在 AES 的 $GF(2^8)$ 中，不可约多项式 $m(x) = x^8 + x^4 + x^3 + x + 1$。计算 $\text{0x87} \cdot \text{0x02}$（即 xtimes($\text{0x87}$)）。
考点乘以 $x$ 时的移位与条件反馈机制。
  1. 乘数 $\text{0x02}$ 的多项式形式即为 $x$。
  2. 将被乘数化为二进制：$\text{0x87} = 1000 0111_2$。
  3. 它的多项式表示为：$x^7 + x^2 + x + 1$。
  4. 普通的乘以 $x$：$x(x^7+x^2+x+1) = x^8 + x^3 + x^2 + x$。
    在二进制中表现为整体左移一位：$0000 1110_2$（溢出了一个1）。
  5. 由于出现了 $x^8$，必须用不可约多项式降次。$x^8 \equiv x^4 + x^3 + x + 1$（十六进制为 $\text{0x1B}$）。
  6. 将剩余部分 $(x^3+x^2+x)$ 与 $(x^4+x^3+x+1)$ 模2相加（异或）：
    $(x^3+x^2+x) + (x^4+x^3+x+1) = x^4 + x^2 + 1$。
  7. 对应的二进制为 $0001 0101_2$，化为十六进制为 $\text{0x15}$。
✅ AES运算口诀：xtimes 操作：将输入左移1位，如果最高位原本是1，则把移位结果与常量 $\text{0x1B}$ 进行 XOR。这不仅是手算技巧，也是芯片里的硬件连线逻辑。
6.7 本原元的生成规律与阶 中等
在 $GF(2^4)$ 中，假设 $x$ 是一根本原多项式的根，故为本原元。求元素 $x^{10}$ 在乘法群中的阶，并说明它能否作为该域的一个新本原元。
考点利用循环群阶公式 $|a^k| = \frac{n}{\gcd(k,n)}$ 求解。群的大小 $n = 2^m - 1$。
  1. 首先确定 $GF(2^4)$ 的乘法群结构。该群剔除了 $0$ 元，大小为 $2^4 - 1 = 15$。
  2. 本原元 $x$ 的阶即为 $15$，它生成群的所有15个元素。
  3. 计算 $x^{10}$ 的阶。使用群论公式：$\text{ord}(x^{10}) = \frac{15}{\gcd(10, 15)} = \frac{15}{5} = 3$。
  4. 验证：$(x^{10})^3 = x^{30} = (x^{15})^2 = 1^2 = 1$。
  5. 结论：$x^{10}$ 的阶为3。因为 $3 \neq 15$，所以它不能生成整个群，不能作为本原元。
✅ 拓展：$GF(2^4)$ 共有 $\varphi(15) = \varphi(3 \times 5) = 2 \times 4 = 8$ 个本原元，分别为 $x^k$，其中 $\gcd(k,15)=1$。
6.8 AES 加法循环矩阵乘法（MixColumns）前奏 中等
在 AES 的列混淆（MixColumns）中，需计算 $GF(2^8)$ 下矩阵乘法，其中一个操作是计算 $\text{0x03} \cdot \text{0xD4}$。基于上题 xtimes，请计算该结果。
考点利用分配律 $\text{0x03} \cdot Y = (\text{0x02} \oplus \text{0x01}) \cdot Y = \text{xtimes}(Y) \oplus Y$ 将复杂乘法拆解为基本操作。
  1. 分解常数乘数：多项式 $x+1$ 对应 $\text{0x03}$，所以 $\text{0x03} \cdot \text{0xD4} = \text{xtimes}(\text{0xD4}) \oplus \text{0xD4}$。
  2. 计算 $\text{xtimes}(\text{0xD4})$：
    $\text{0xD4} = 1101 0100_2$。最高位为1。
    左移一位：$1010 1000_2$（溢出被丢弃）。
    与 $\text{0x1B} = 0001 1011_2$ 异或：
    $1010 1000 \oplus 0001 1011 = 1011 0011_2 = \text{0xB3}$。
  3. 计算最终和：
    $\text{0xB3} \oplus \text{0xD4}$
    $= 1011 0011_2 \oplus 1101 0100_2 = 0110 0111_2$
  4. 化为十六进制：$0110 \to 6$，$0111 \to 7$。
  5. 结论：计算结果为 $\text{0x67}$。
✅ 总结：在AES中，乘以大常数（如解密时的0x0E等）总是通过分解为若干次 xtimes 并 XOR 叠加实现。这种设计极具效率。
6.9 GF(p^m) 中的费马小定理 中等
证明：对有限域 $GF(q)$（$q=p^m$）中的任意非零元素 $a$，都有 $a^{q-1} = 1$；对所有元素（含0）都有 $a^q = a$。
考点这直接就是拉格朗日定理在有限域乘法群上的结论。
  1. 域 $GF(q)$ 的乘法群（由所有非零元素构成）的阶为 $q-1$。
  2. 由拉格朗日定理的推论，群中任何元素的群阶次幂等于单位元。
  3. 因此对任意非零 $a$，立刻有 $a^{q-1} = 1$。
  4. 在等式两边同乘 $a$，得 $a^q = a$。由于此等式在 $a=0$ 时（$0^q = 0$）显然也成立。
  5. 结论：故对全域所有元素都有 $a^q = a$。
✅ 总结：这其实就是费马小定理从整数域（一维）向广义有限域（多维）的自然推广。这个公式是极其强大的恒等变换工具。
6.10 扩展欧几里得求有限域逆元初步 综合
在 $GF(2^3)$ 中（模多项式 $P(x) = x^3+x+1$），求元素 $a(x) = x^2$ 的乘法逆元。
考点利用多项式欧几里得算法求解 $a(x)u(x) + P(x)v(x) = 1$。
  1. 初始状态：除数 $P(x) = x^3+x+1$，被除数 $a(x) = x^2$。
  2. 第一步相除：$(x^3+x+1) \div x^2$
    商为 $x$，减去 $x \cdot x^2 = x^3$。
    余数为 $x+1$。
    这可以用等式写为：$x+1 = P(x) - x \cdot a(x)$。在 $GF(2)$ 中减号即加号，即 $x+1 = P(x) + x \cdot a(x)$。
  3. 第二步相除：用上一步的除数 $x^2$ 除以余数 $x+1$。
    $(x^2) \div (x+1)$：
    商为 $x+1$。相乘得 $(x+1)^2 = x^2+1$。
    减去得余式：$x^2 - (x^2+1) = 1$。
  4. 将余式1用前面步骤回代：
    $1 = x^2 + (x+1)(x+1)$
    代入 $(x+1) = P(x) + x \cdot a(x)$：
    $1 = a(x) + (x+1)[P(x) + x \cdot a(x)]$（注意首项的 $x^2$ 就是 $a(x)$）
    $= a(x) + (x+1)P(x) + (x^2+x)a(x)$
    合并 $a(x)$ 的系数：$1 + (x^2+x) = x^2+x+1$。
  5. 所以 $a(x)$ 的逆元为 $x^2+x+1$。
✅ 验证：计算 $x^2 \cdot (x^2+x+1) = x^4+x^3+x^2$。使用 $x^3 \equiv x+1$ 和 $x^4 \equiv x^2+x$ 进行降次：
$(x^2+x) + (x+1) + x^2 = x^2+x^2+x+x+1 = 1$ ✓。这正是 AES S盒的核心操作。
6.11 用指数法求有限域逆元 综合
基于题6.9中的费马小定理推广，给出一个在 $GF(p^m)$ 中求解逆元的不使用欧几里得算法的直接计算公式。
考点$a^{q-1} = 1 \Rightarrow a \cdot a^{q-2} = 1$。
  1. 在 $GF(p^m)$ 中，大小 $q = p^m$。
  2. 对于任意非零元素 $a$，其属于阶为 $q-1$ 的乘法群。
  3. 由此恒有等式：$a^{p^m - 1} = 1$。
  4. 将指数拆分出一个一次项：$a \cdot a^{p^m - 2} = 1$。
  5. 根据逆元的定义 $a \cdot a^{-1} = 1$，显然括号里的部分就是逆元。
  6. 结论：在 $GF(p^m)$ 中，逆元的计算公式直接是 $a^{-1} = a^{p^m - 2}$。
✅ 总结：在 $GF(2^8)$ 中求逆元，只需计算 $a^{254}$ 即可。这可以通过快速幂算法在确定的时钟周期内完成（比如抵抗侧信道攻击时比时间可变的欧几里得法更受青睐）。这被硬件设计者广泛采用。
6.12 GF(q) 的子域结构 综合
有限域 $GF(2^6)$ 可能包含哪些子域？
考点子域存在定理：$GF(p^m)$ 包含 $GF(p^k)$ 当且仅当 $k$ 整除 $m$。
  1. 首先考察基础特征 $p=2$，扩张次数 $m=6$。
  2. 根据定理，必须找出 6 的所有正约数 $k$。
  3. $m=6$ 的约数有：$1, 2, 3, 6$。
  4. 对应的子域大小为：
    $k=1 \Rightarrow GF(2^1) = GF(2)$（基域）
    $k=2 \Rightarrow GF(2^2) = GF(4)$
    $k=3 \Rightarrow GF(2^3) = GF(8)$
    $k=6 \Rightarrow GF(2^6)$（自身）
  5. 结论：$GF(2^6)$ 含有 $GF(2), GF(4), GF(8)$ 三个真子域。
✅ 总结：这在代数设计中极其有用，例如把复杂的 $GF(2^8)$ 乘法投影到它的子域 $GF(2^4)$（即把大域看作小域的二次扩张）可以极大减少硬件电路面积，这是 AES 硬件实现的最高端技巧（复合域技术）。
6.13 特征性质的深度应用 综合
证明在 $GF(2^m)$ 中，$(x+y)^2 = x^2 + y^2$ 恒成立。这被称为"大一新生的梦想"。
考点二项式展开与有限域特征的关系。
  1. 使用二项式定理对多项式展开：$(x+y)^2 = x^2 + 2xy + y^2$。
  2. 在 $GF(2^m)$ 中，底层基域是 $GF(2)$。
  3. 系数 $2$ 实际上代表 $1+1$。
  4. 由于特征为2，域中恒有 $1+1=0$，因此系数 $2 \equiv 0$。
  5. 交叉项 $2xy = 0 \cdot xy = 0$ 彻底消失。
  6. 结论：$(x+y)^2 = x^2 + y^2$。
✅ 拓展：更一般地，在任何特征为 $p$ 的域 $GF(p^m)$ 中，恒有 $(x+y)^{p^k} = x^{p^k} + y^{p^k}$。这意味着操作映射 $\phi(x) = x^p$ 保留了加法，它是一个环自同态，称为弗罗贝尼乌斯自同态（Frobenius Endomorphism）。
6.14 本原多项式的非等价性 综合
已知 $x^3+x+1$ 和 $x^3+x^2+1$ 都是 $GF(2)$ 上的本原多项式，它们各自构建出来的两个 $GF(2^3)$ 域相同吗？
考点同构唯一性定理：使用不同本原多项式构建的域在符号层面上可能不同，但代数结构完全同构。
  1. 令域 $F_1 = GF(2)[x]/(x^3+x+1)$，生成元为 $\alpha$。则 $\alpha^3 = \alpha + 1$。
  2. 令域 $F_2 = GF(2)[y]/(y^3+y^2+1)$，生成元为 $\beta$。则 $\beta^3 = \beta^2 + 1$。
  3. 考虑在 $F_1$ 中寻找元素 $z$，使得 $z$ 满足 $F_2$ 的构造方程，即 $z^3+z^2+1=0$。
  4. 尝试 $z = \alpha+1$：
    $z^3 = (\alpha+1)^3 = \alpha^3+\alpha^2+\alpha+1$
    代入 $\alpha^3=\alpha+1$ 得 $z^3 = (\alpha+1)+\alpha^2+\alpha+1 = \alpha^2$
    计算方程 $z^3+z^2+1 = \alpha^2 + (\alpha+1)^2 + 1 = \alpha^2 + (\alpha^2+1) + 1 = 0$！
  5. 由此找到了将 $F_2$ 中的 $\beta$ 映射到 $F_1$ 中的 $\alpha+1$ 的双射同态。
  6. 结论：在代数内部结构上，它们是百分之百等同（同构）的，只是对于元素的命名和乘法规则做了坐标系变换。
✅ 总结：在工程上，选择不同的多项式只是为了底层硬件连线和查表逻辑的优化考量，AES选用 $x^8+x^4+x^3+x+1$ 并非因为它是唯一的，而是因为这个多项式的运算速度和电路复杂度取得了极佳的平衡。
6.15 AES S-盒构造代数解析 综合
AES 的字节代换（SubBytes）步骤首先求输入字节在 $GF(2^8)$ 中的乘法逆元（若是 $\text{0x00}$ 则映射自身），然后再乘上一个固定的矩阵。为什么这个简单的代数操作能提供如此强的抗差分密码分析能力？
考点有限域逆元映射 $x \mapsto x^{-1}$（或 $x^{254}$）的极高代数次数和极强非线性。
  1. 首先，仿射变换（乘以矩阵和异或常数）只提供位上的线性混淆，没有任何抗差分强度。安全防线全在求逆元这一步。
  2. 分析逆元映射：$f(x) = x^{-1} = x^{254}$。
  3. 在特征为2的域中，通过布尔函数的代数次数分析发现，由于多项式 $x^{254}$ 的度极高且各项系数分布稠密（用香农理论叫做混淆度高），它的各项位之间的联动非常错综复杂。
  4. 计算其差分均匀度（Differential Uniformity），结果为4，这是8比特可逆映射（S盒）所能达到的最优或极优值。这确保了输入差分不会高概率地传播到输出差分。
  5. 为了破坏代数攻击（避免公式形式过于简单），最后乘以一个特定的矩阵将 $GF(2^8)$ 内紧凑的代数关系拆解为跨域的混乱位关系。
  6. 结论：有限域的"除法"（逆元）是一把在混乱与数学美感之间取得极佳平衡的完美利刃。
✅ 总结：AES 算法之所以取代了由于设计靠直觉而有些混乱的 DES，正是因为它全身都是由完美、对称的代数结构尤其是有限域多项式拼接而成。它的美感与安全性同步达到了巅峰。
