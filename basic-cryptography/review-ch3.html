第3章：多项式理论

📖 理论公式与深度解读

1. 多项式带余除法

公式推导： 对域 $F$ 上的多项式 $f, g(g \neq 0)$，存在唯一 $q, r$，使得 $f(x) = q(x)g(x) + r(x)$，且 $\deg(r) < \deg(g)$。

适用场景： 用于 $GF(p^m)$ 有限域中的元素乘法降次操作（如 AES 的 MixColumns）。余式 $r(x)$ 即为模 $g(x)$ 的结果。

2. 因式定理 (Factor Theorem)

公式推导： $(x-\alpha) \mid f(x) \iff f(\alpha) = 0$。

适用场景： 试根法寻找一次因式的理论基础，特别用于判定 2、3 次多项式是否不可约（无根即不可约）。

3. 不可约多项式判定 (Eisenstein Criterion)

公式推导： 存在素数 $p$ 满足：$p \nmid a_n$，$p \mid a_i$ ($i

适用场景： 用于有理数域上的高次多项式不可约性判定，避免复杂的待定系数法。

🧠 配套计算类真题解析 (共15题)

题3.1：多项式余数计算
求 $f(x) = x^4+3x^2+1$ 除以 $x-2$ 的余数。
🎯 核心考点：余数定理（多项式除以一次因式 $x-a$ 的余式等于 $f(a)$）。
📝 题型拆解：计算题。通过直接代入求值计算余数，避免进行复杂的多项式长除法。
🛠️ 详细解题步骤：
  1. 根据余数定理，多项式 $f(x)$ 除以 $x-a$ 的余数等于多项式在 $x=a$ 处的值。本题中除式为 $x-2$，所以 $a=2$。
  2. 将 $x=2$ 代入目标多项式 $f(x)$ 中计算：$f(2) = 2^4 + 3(2^2) + 1$。
  3. 分布计算各项的值：$2^4 = 16$，$3 \times 4 = 12$，常数项为 $1$。
  4. 求和：$f(2) = 16 + 12 + 1 = 29$。
  5. 因此，所求的余数为 29。
✅ 校验与总结：检查代数运算无误。余数定理是求解一次因式除法余数的最优解法，切忌在此类题目中浪费时间进行多项式长除。
题3.2：带余除法完整长除
在 $\mathbb{R}[x]$ 中，用 $x^3+2x+1$ 除以 $x^2+1$ 求商和余式。
🎯 核心考点：多项式带余除法（长除法）及其唯一性定理。
📝 题型拆解：计算题。要求使用标准的多项式长除法，逐次消去被除数的最高次项，直到余式的次数严格小于除式的次数。
🛠️ 详细解题步骤：
  1. 观察被除数 $f(x) = x^3+2x+1$ 的最高次项 $x^3$，以及除数 $g(x) = x^2+1$ 的最高次项 $x^2$。
  2. 第一步消除最高次项：用 $x^3$ 除以 $x^2$ 得到商的第一项 $x$。
  3. 用被除数减去商的第一项与除数的乘积：$(x^3+2x+1) - x(x^2+1) = (x^3+2x+1) - (x^3+x) = x+1$。此时得到的中间余式为 $x+1$。
  4. 判断是否终止：当前余式 $x+1$ 的次数为 1，而除数 $x^2+1$ 的次数为 2。因为 $1 < 2$，满足带余除法终止条件 $\deg(r) < \deg(g)$。
  5. 得出结论：商 $q(x) = x$，余式 $r(x) = x+1$。
✅ 校验与总结：验证 $q(x)g(x) + r(x) = x(x^2+1) + (x+1) = x^3+x+x+1 = x^3+2x+1$，结果与原多项式一致，计算正确。长除法要特别注意对齐项的幂次并小心减法运算中的符号。
题3.3：GF(2) 中的多项式运算
在 $GF(2)$ 中计算 $(x^3+x) + (x^3+x^2)$。
🎯 核心考点：有限域 $GF(2)$ 上的多项式加减法法则（系数模 2 加法，等价于异或运算）。
📝 题型拆解：计算题。将同类项合并时，系数进行模 2 运算，由于 $1+1 \equiv 0 \pmod 2$，相同项相加直接抵消。
🛠️ 详细解题步骤：
  1. 写出多项式加法的按项组合形式：$(x^3 + x^3) + x^2 + x$。
  2. 在普通的环中，计算结果应为 $2x^3 + x^2 + x$。
  3. 但本题在 $GF(2)$ 域上，所有多项式系数都必须模 2。计算系数的模 2 结果：同类项 $x^3$ 的系数为 $1+1 = 2 \equiv 0 \pmod 2$。
  4. 因此，$x^3$ 项被完全抵消，剩下的项其系数均为 1。
  5. 最终化简结果为：$x^2+x$。
✅ 校验与总结：在 $GF(2)$ 中，加法与减法完全等价，相同项直接消除（即异或抵消原理）。这是密码学（如 AES 算法）中最基本、最频繁操作的多项式运算基石。
题3.4：因式分解未知数
若 $x+1$ 是 $x^3+kx^2+2$ 的因式，求 $k$。
🎯 核心考点：因式定理及其逆向应用（如果 $x-a$ 是 $f(x)$ 的因式，则 $a$ 必定是多项式的根，即 $f(a)=0$）。
📝 题型拆解：计算题。通过利用因式定理建立关于参数 $k$ 的一元一次方程来求解参数值。
🛠️ 详细解题步骤：
  1. 根据因式定理，因为 $x+1$（可以写作 $x-(-1)$）是多项式 $f(x) = x^3+kx^2+2$ 的一个因式，所以 $x=-1$ 必定是该多项式的根。
  2. 代入根：由 $f(-1) = 0$，得到方程 $(-1)^3 + k(-1)^2 + 2 = 0$。
  3. 计算各次幂：$(-1)^3 = -1$，$(-1)^2 = 1$。
  4. 化简方程为：$-1 + k \cdot 1 + 2 = 0$，即 $k + 1 = 0$。
  5. 移项求解得到：$k = -1$。
✅ 校验与总结：将 $k=-1$ 代回原多项式得到 $x^3-x^2+2$。当 $x=-1$ 时，$-1 - 1 + 2 = 0$，校验通过。利用因式定理求未知系数是多项式章节的高频经典题型。
题3.5：不可约判定（试根法）
判断 $x^3+x^2+1$ 在 $GF(2)$ 上是否可约。
🎯 核心考点：低次多项式（2 次或 3 次）不可约性的试根判定法。
📝 题型拆解：证明/判定题。核心定理：域上次数为 2 或 3 的多项式可约当且仅当它在该域中有根。
🛠️ 详细解题步骤：
  1. 分析次数：目标多项式 $f(x) = x^3+x^2+1$ 的次数为 3。根据多项式因式分解原理，如果一个 3 次多项式可约，它必然可以分解为一个 1 次因式和一个 2 次因式，或者三个 1 次因式的乘积。这意味着它必然包含至少一个 1 次因式。
  2. 根据因式定理，包含 1 次因式等价于在有限域中有根。
  3. 在 $GF(2)$ 中，可能的元素只有 $0$ 和 $1$。我们使用试根法逐近代入检验。
  4. 代入 $x=0$：$f(0) = 0^3 + 0^2 + 1 = 1 \neq 0$，说明 0 不是根。
  5. 代入 $x=1$：$f(1) = 1^3 + 1^2 + 1 = 1+1+1 = 3 \equiv 1 \pmod 2 \neq 0$，说明 1 也不是根。
  6. 由于多项式在 $GF(2)$ 中没有根，且其次数为 3，因此该多项式在 $GF(2)$ 上不可约。
✅ 校验与总结：判定逻辑清晰严密。试根法对于次数 $\ge 4$ 的多项式不一定适用（例如可能分解为两个二次不可约多项式的乘积但无一次因式无根），但对 2、3 次多项式是充要条件，这是极易混淆的陷阱。
题3.6：艾森斯坦判别
判断 $x^4+6x^3+12x+6$ 在 $\mathbb{Q}$ 上是否可约。
🎯 核心考点：艾森斯坦判别法（Eisenstein's Criterion），用于判定有理数域 $\mathbb{Q}$ 上的高次多项式是否不可约。
📝 题型拆解：证明/判定题。通过寻找一个满足特定整除条件的素数 $p$，直接得出在 $\mathbb{Q}$ 上的不可约性结论。
🛠️ 详细解题步骤：
  1. 写出多项式各项系数：$a_4 = 1, a_3 = 6, a_2 = 0, a_1 = 12, a_0 = 6$。
  2. 目标是寻找一个素数 $p$ 满足三个条件：(1) $p \nmid a_n$；(2) 对所有 $i < n$，有 $p \mid a_i$；(3) $p^2 \nmid a_0$。
  3. 尝试素数 $p=3$：
    - 最高次系数 $a_4=1$，$3 \nmid 1$，满足条件(1)；
    - 其他系数为 $6, 0, 12, 6$，都能被 3 整除，满足条件(2)；
    - 常数项 $a_0 = 6$，$p^2 = 9$，$9 \nmid 6$，满足条件(3)。
  4. 或者尝试素数 $p=2$：$2 \nmid 1$；$2$ 整除 $6, 0, 12, 6$；且 $2^2 = 4 \nmid 6$。同样符合判别法条件。
  5. 由于能找到满足艾森斯坦判别法条件的素数，故断定多项式在 $\mathbb{Q}$ 上不可约。
✅ 校验与总结：艾森斯坦判别法是充分不必要条件，能找到这样的素数必定不可约，但找不到不代表可约。本题中素数 2 或 3 均可成功判定，考试时只需找出一个证明即可。
题3.7：多项式的最大公因式
求 $\gcd(x^3-1, x^2-1)$。
🎯 核心考点：多项式的最大公因式求解（通过因式分解法或辗转相除法）。
📝 题型拆解：计算题。多项式次数较低且易于分解，采用因式分解法能最快得出结果，避免使用容易出错的辗转相除。
🛠️ 详细解题步骤：
  1. 首先对第一个多项式进行因式分解：使用立方差公式，$x^3-1 = (x-1)(x^2+x+1)$。由于 $x^2+x+1$ 在实数域上判别式 $\Delta < 0$，不可再分解。
  2. 对第二个多项式进行因式分解：使用平方差公式，$x^2-1 = (x-1)(x+1)$。
  3. 比较两个多项式的全部分解因式：$x^3-1$ 包含 $(x-1)$ 和 $(x^2+x+1)$；$x^2-1$ 包含 $(x-1)$ 和 $(x+1)$。
  4. 提取公共因式：两者的唯一公共因式为 $(x-1)$。
  5. 所以它们的最大公因式 $\gcd(x^3-1, x^2-1) = x-1$。
✅ 校验与总结：若使用辗转相除法：$(x^3-1) = x(x^2-1) + (x-1)$，接着 $(x^2-1) = (x+1)(x-1) + 0$，最后非零余式就是 $x-1$，两种方法结果一致。对低次特征多项式，因式分解法更具效率。
题3.8：多重根判定
判断 $x=-1$ 是 $x^4+2x^3+2x^2+2x+1$ 的几重根？
🎯 核心考点：多项式多重根的求导判定法（$x=a$ 是 $k$ 重根，当且仅当它使原多项式及前 $k-1$ 阶导数值为零，而第 $k$ 阶导数值非零）。
📝 题型拆解：计算题。通过逐次求导并代入给定的根进行检验，来确认重数。
🛠️ 详细解题步骤：
  1. 设原多项式为 $f(x) = x^4+2x^3+2x^2+2x+1$。首先验证其是否为根：$f(-1) = 1 - 2 + 2 - 2 + 1 = 0$。确认是根。
  2. 求一阶导数：$f'(x) = 4x^3+6x^2+4x+2$。代入 $x=-1$ 得 $f'(-1) = -4 + 6 - 4 + 2 = 0$。一阶导为0，说明其重数至少为 2。
  3. 求二阶导数：$f''(x) = 12x^2+12x+4$。代入 $x=-1$ 得 $f''(-1) = 12(1) - 12 + 4 = 4$。
  4. 因为 $f(-1)=0$，$f'(-1)=0$，但 $f''(-1) \neq 0$，所以 $x=-1$ 正好是 2 重根。
✅ 校验与总结：可以通过综合除法验证：$f(x) \div (x+1) = x^3+x^2+x+1$，再除以 $(x+1)$ 得 $x^2+1$（不再含 $x+1$ 因子）。因此 $f(x) = (x+1)^2(x^2+1)$，准确印证了 $x=-1$ 为二重根。求导法显著降低了代数除法的复杂性。
题3.9：多项式互素组合
若 $\gcd(f,g)=1$，且 $f \mid h$，$g \mid h$，证明 $fg \mid h$。
🎯 核心考点：多项式的裴蜀定理（Bézout's identity）与整除性质推导。
📝 题型拆解：证明题。需利用互素条件构造出乘积恒等式，再结合整除定义进行代数恒等变形证明。
🛠️ 详细解题步骤：
  1. 根据互素条件 $\gcd(f,g)=1$，由多项式裴蜀定理可知，必定存在多项式 $u(x), v(x)$ 满足等式：$u(x)f(x) + v(x)g(x) = 1$。
  2. 对上述等式两边同时乘以多项式 $h(x)$，得到：$u(x)f(x)h(x) + v(x)g(x)h(x) = h(x)$。
  3. 分析整除条件：已知 $g \mid h$，所以可以设 $h = k_1 g$。将其代入第一项得：$ufh = uf(k_1 g) = k_1(fg)$。显然，这一项能被 $fg$ 整除。
  4. 同理，已知 $f \mid h$，设 $h = k_2 f$。代入第二项得：$vgh = vg(k_2 f) = k_2(fg)$。这一项也能被 $fg$ 整除。
  5. 等式左边的两部分都含有公因式 $fg$，因此它们的和 $h(x)$ 必然也能被 $fg$ 整除，即证明了 $fg \mid h$。
✅ 校验与总结：这道题是极其经典的代数结构基础证明。逻辑链条为：互素 $\implies$ 裴蜀等式 $\implies$ 同乘待证项 $\implies$ 替换整除因子导出结论。记住这一范式，任何互素推导题都能迎刃而解。
题3.10：GF(3) 中的分解
在 $\mathbb{Z}_3$ 中分解 $x^3+2x$。
🎯 核心考点：有限域 $\mathbb{Z}_p$ （这里 $p=3$）下多项式的因式分解技巧（系数的同余代换）。
📝 题型拆解：计算题。通过提取公因式并灵活应用模 $p$ 环境下的同余等价关系，将多项式分解为一次因式的乘积。
🛠️ 详细解题步骤：
  1. 首先观察多项式 $x^3+2x$，每一项都含有变量 $x$，可以直接提取出公因式 $x$：$x^3+2x = x(x^2+2)$。
  2. 接下来需要在 $\mathbb{Z}_3$ 域中处理二次因式 $x^2+2$。在模 3 的环境下，数字 $2$ 与 $-1$ 是同余的，即 $2 \equiv -1 \pmod 3$。
  3. 利用这一同余性质替换常数项，得到：$x^2+2 \equiv x^2-1 \pmod 3$。
  4. 此时多项式变为了熟知的平方差形式，使用基本公式分解：$x^2-1 = (x-1)(x+1)$。
  5. 综合起来，得到最终的完全分解式：$x(x-1)(x+1)$。在 $\mathbb{Z}_3$ 中 $x-1$ 也可写为 $x+2$，即 $x(x+1)(x+2)$。
✅ 校验与总结：验证根的情况：代入 0 得 0；代入 1，$1+2=3\equiv 0$；代入 2，$8+4=12\equiv 0$。三个元素全是根，说明必可分解为 $x(x-1)(x-2)$ 即 $x(x+2)(x+1)$，分解结果准确无误。模运算中正负代换是破题关键。
题3.11：构造有限域维数
不可约多项式 $P(x)$ 必须是什么次数才能构造 $GF(2^8)$？
🎯 核心考点：有限扩域 $GF(p^n)$ 的代数构造定理。
📝 题型拆解：概念解析题。只需套用有限域生成的维度理论，明确扩域指数与不可约多项式次数之间的对应关系。
🛠️ 详细解题步骤：
  1. 根据抽象代数理论，要构造有限域 $GF(p^n)$，基础域为 $\mathbb{Z}_p$。
  2. 构造的方法是：在多项式环 $\mathbb{Z}_p[x]$ 中选取一个次数恰好为 $n$ 的不可约多项式 $P(x)$。
  3. 取商环 $\mathbb{Z}_p[x] / (P(x))$，该商环的维度即为 $n$，形成包含 $p^n$ 个元素的域。
  4. 本题中要求构造的域是 $GF(2^8)$，对应的素底数为 $p=2$，扩域指数为 $n=8$。
  5. 因此，必须寻找一个在 $GF(2)$ 上不可约的 $8$ 次多项式来作为模多项式（例如 AES 中使用的是 $x^8+x^4+x^3+x+1$）。
✅ 校验与总结：这是一个基础且重要的结论：$GF(p^n)$ 必须依赖 $n$ 次不可约多项式来建立。如果次数选错，不仅生成空间大小不对，且如果选用了可约多项式，生成的代数结构将包含零因子，无法构成域。
题3.12：有理根定理
多项式 $2x^3+x-1=0$ 的有理根候选有哪些？
🎯 核心考点：整系数多项式的有理根判定定理（Rational Root Theorem）。
📝 题型拆解：计算题。通过提取最高次项系数和常数项的全部因子，交叉组合列举出所有可能的有理根。
🛠️ 详细解题步骤：
  1. 有理根定理指出：如果整系数多项式 $a_nx^n + \dots + a_0 = 0$ 有有理根 $p/q$（其中 $p, q$ 互素），那么 $p$ 必须整除常数项 $a_0$，$q$ 必须整除最高次项系数 $a_n$。
  2. 对于目标多项式 $2x^3+x-1=0$：常数项 $a_0 = -1$，最高次系数 $a_3 = 2$。
  3. 求 $p$ 的可能取值：$p$ 需要是 $-1$ 的因数，所以 $p \in \{1, -1\}$。
  4. 求 $q$ 的可能取值：$q$ 需要是 $2$ 的正因数，所以 $q \in \{1, 2\}$。
  5. 组合所有可能的 $p/q$ 形式，得到候选的有理根集合：$\{ \pm \frac{1}{1}, \pm \frac{1}{2} \}$。
  6. 去重并简化后，候选列表为：$1, -1, 1/2, -1/2$。
✅ 校验与总结：有理根定理只提供“候选池”，并非每个候选都是真正的根。比如本题中实际代入检验可知 $x=1/2$ 时 $2(1/8)+1/2-1 = -1/4 \neq 0$；而其他候选也不是根（实际上此方程在有理数内无解）。寻找候选根是因式分解高次方程的第一步。
题3.13：GF(2)下多项式互素
判断 $GF(2)$ 下 $x^2+1$ 和 $x^2+x+1$ 互素吗。
🎯 核心考点：有限域 $GF(2)$ 中的多项式不可约性与最大公因式判定。
📝 题型拆解：判断题。通过分别对两个多项式进行因式分解或试根，分析它们是否包含公共的不可约因子。
🛠️ 详细解题步骤：
  1. 先分析第一个多项式 $f(x) = x^2+1$。在 $GF(2)$ 下，由于 $1+1=0$，它满足完全平方公式特征：$x^2+1 = x^2+2x+1 = (x+1)^2$。这说明其唯一的不可约因式是 $(x+1)$。
  2. 再分析第二个多项式 $g(x) = x^2+x+1$。它是一个 2 次多项式，尝试使用试根法判断是否可约。
  3. 代入 0 得：$g(0) = 0+0+1=1 \neq 0$。代入 1 得：$g(1) = 1+1+1=3 \equiv 1 \pmod 2 \neq 0$。
  4. 既然 $g(x)$ 无根，它在 $GF(2)$ 下是不可约多项式。
  5. 比较两者的因子：$f(x)$ 仅包含因子 $(x+1)$，而 $g(x)$ 是整体不可约的。
  6. 两者没有任何公共的因式，因此最大公因式为 1，结论是互相素（互素）。
✅ 校验与总结：也可直接应用辗转相除法：$(x^2+x+1) = 1 \cdot (x^2+1) + x$。接着 $(x^2+1) = x \cdot x + 1$。最后余数是 1，直接证明 $\gcd = 1$。对于次数低的题，直接因式结构分析更加直观且不容易计算出错。
题3.14：因式定理与复数
若 $1+i$ 是实系数方程 $f(x)=0$ 的根，还能确定什么因式？
🎯 核心考点：实系数多项式的复根共轭成对定理。
📝 题型拆解：推导题。利用定理直接推导出隐含的共轭根，并通过两个一次复因式的相乘，构造出一个二次实系数因式。
🛠️ 详细解题步骤：
  1. 明确前提条件：多项式 $f(x)$ 的系数全部为实数。
  2. 应用定理：代数基本定理的推论指出，对于实系数多项式，如果有一个复数根 $z = a+bi$，那么它的共轭复数 $\bar{z} = a-bi$ 必然也是方程的根。
  3. 已知 $1+i$ 是根，因此其共轭复数 $1-i$ 必定也是根。
  4. 根据因式定理，$f(x)$ 必然同时包含因式 $(x - (1+i))$ 和 $(x - (1-i))$。
  5. 将这两个一次因式相乘，构造出二次因式：$(x - 1 - i)(x - 1 + i)$。
  6. 使用平方差公式展开：$((x-1) - i)((x-1) + i) = (x-1)^2 - i^2 = (x^2 - 2x + 1) - (-1) = x^2 - 2x + 2$。
  7. 因此，可以确定 $f(x)$ 必然包含实系数的二次因式 $(x^2-2x+2)$。
✅ 校验与总结：这是降次解高阶实系数方程的关键突破口。一旦获知一个复根，立刻能构造出一个实系数二次因式，通过多项式长除法 $f(x) \div (x^2-2x+2)$ 即可把方程降两阶，简化后续求根难度。
题3.15：代数基本定理结合性
所有实系数奇次多项式必有实根，为什么？
🎯 核心考点：代数基本定理与复根成对定理的综合逻辑推导。
📝 题型拆解：证明/简答题。从复数域的封闭性出发，结合复根对称性规律，用反证或奇偶性分析直接得出结论。
🛠️ 详细解题步骤：
  1. 根据代数基本定理，一个 $n$ 次的复系数多项式在复数域中必定有且仅有 $n$ 个根（计算重数）。因此，一个奇数次多项式在复数域中必定有奇数个根。
  2. 根据复根共轭成对定理，由于该多项式是实系数的，它的所有非实数复根（虚根）必须以共轭对（即 $a+bi$ 和 $a-bi$）的形式成对出现。
  3. 这意味着虚根的总数量必定是一个偶数（比如 0个，2个，4个...）。
  4. 多项式的总根数 = 实根数量 + 虚根数量。
  5. 已知总根数 $n$ 为奇数，虚根数量为偶数。由奇偶性规律（奇数 = 实数 + 偶数），实根数量必定为奇数。
  6. 因为实根数量是奇数，即实根数量 $\ge 1$，所以它至少存在一个实数根。
✅ 校验与总结：从微积分连续性（介值定理）角度也能完美证明：由于最高次项为奇数，$x \to +\infty$ 时 $f(x)$ 与 $x \to -\infty$ 时的符号必然相反，连续函数穿过 x 轴必然至少产生一个零点。两种证法殊途同归。
