1. 代数结构与初步概念

本章主要介绍了抽象代数的最基本单元：代数结构。这是后续群、环、域等概念的基石，核心在于理解集合与运算的“封闭性”。

  • 代数结构由一个**非空集合 M** 和其上的**代数运算**组成。
  • **封闭性**：对于集合 M 中的任意两个元素 a, b，经过运算后的结果 c 必定也属于集合 M。
  • 举例：整数集合对加法封闭，但奇数集合对加法不封闭（奇数+奇数=偶数）。

置换的本质是双射函数。两个置换相乘即为函数的合成 f(g(x))。

注意：计算时通常遵循从右向左的复合顺序。

示例：
p1 = (1 2 3)
p2 = (2 3)
p1 * p2 计算：
1 -> [p2] -> 1 -> [p1] -> 2
2 -> [p2] -> 3 -> [p1] -> 1
3 -> [p2] -> 2 -> [p1] -> 3
结果: (1 2)

2. 初等数论

数论是RSA等非对称加密体系的灵魂。本节包含必须掌握的核心计算，包括最大公因子求解、素数分解、同余类以及欧拉函数的计算。

  • **算术基本定理**：任何大于1的整数都能唯一地分解为素数的乘积（不计顺序）。这为求欧拉函数提供了基础。
  • **同余 (Congruence)**：a ≡ b (mod m) 意味着 m 整除 (a-b)。
  • **欧拉函数 φ(n)**：小于或等于n且与n互素的正整数个数。公式：如果 n = p1k1 p2k2，则 φ(n) = n(1 - 1/p1)(1 - 1/p2)。

3. 多项式理论

多项式理论是构造高级有限域（如AES使用的有限域）的基础。重点在于区分可约与不可约多项式，并能利用定理判断因式。

可约性 (Irreducibility)

不可约多项式类似于整数中的素数。如果一个次数大于0的多项式不能写成两个次数较低的多项式的乘积，它就是不可约的（素多项式）。这取决于所讨论的域（如 $x^2+1$ 在实数域不可约，在复数域可约）。

核心定理 2.33

多项式 f(x) 含有根 α（即 f(α) = 0），当且仅当 (x - α) 是 f(x) 的一个因式。

考试高频考点：利用此定理求证或提取一次因式。

4. 群论

群是定义了单一运算并满足特定公理的代数结构。掌握群的四个公理、元素的阶以及拉格朗日定理是解题的钥匙。

群的四条公理

1. 封闭性 2. 结合律 3. 存在单位元 e 4. 存在逆元 a-1

（若额外满足交换律，则称为阿贝尔群/交换群）

元素的阶 (Order): 满足 ak = e 的最小正整数 k。在有限群中，元素的阶必为有限。

循环群: 若群 G 中的所有元素都可以表示为某个元素 g 的幂次（gn），则 G 是循环群，g 称为生成元。

定理内容：

对于有限群 G 的任何子群 H，子群 H 的阶（元素个数）必定能整除群 G 的阶。

极其重要的推论：

  • 群中任意元素的阶必须整除群的阶。
  • 素数阶的群必定是循环群。

5. 环与域

本章进一步增加了结构约束。环定义了两种运算（通常记为加和乘），而域是结构最完善的代数系统，支持加减乘除四则运算。

概念递进关系

环 (Ring) → 存在加法群且乘法满足结合律与分配律。
整环 (Integral Domain) → 交换环且无零因子。
域 (Field) → 交换除环（非零元素都有乘法逆元）。

模 n 剩余类环 Zn 的性质 (常考)

  • 运算：(a+b) mod n, (a*b) mod n
  • 是否为域： 当且仅当 n 为素数时，Zn 才是域 (即 GF(p) 或 Zp)。
  • 可逆元： 与 n 互素的元素具有乘法逆元。
  • 零因子： 与 n 不互素的非零元素。例如在 Z6 中，2*3 = 0，故2和3都是零因子。

6. 有限域 (高级密码学基石)

这部分是考试压轴题常客，主要涉及从低阶有限域扩域到高阶有限域的计算，特别是如何利用不可约多项式构造 GF(pm)。

构造法核心口诀

要构造有限域 GF(pm)，你需要：
1. 找到基础素数域 GF(p)（通常为二元域 GF(2)）。
2. 找到一个在 GF(p) 上的 m 次不可约多项式 f(x)。
3. 该有限域的元素是次数小于 m 的所有多项式。
4. 加减法对应系数在 GF(p) 下运算；乘法为多项式相乘后对 f(x) 取余（即令 f(x)=0 进行降次）。

📝 元素的不同表示形式

在 GF(2m) 中，同一元素有四种等价写法，掌握其转化对计算至关重要：

表示法 示例 (GF(2^3))
多项式 x2 + 1
二进制 101 (对应多项式系数)
整数 5 (101的十进制)
指数形式 gk (如果g是本原元)
