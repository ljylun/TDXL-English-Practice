3.1 数域的定义与多项式环层级

数域是多项式理论赖以生存的土壤。了解多项式系数所在的域，是判断多项式性质的第一步。

关联知识点：域的公理体系与环的区别、有限域扩张的构造

域（Field）与多项式环（Polynomial Ring）

域：允许无障碍进行加、减、乘、除（除数不为0）的代数结构。经典数域层级：$\mathbb{Q} \subset \mathbb{R} \subset \mathbb{C}$。

多项式环 $F[x]$：系数取自域 $F$ 的所有多项式 $f(x) = a_nx^n + \dots + a_1x + a_0$ 构成的集合，对加法和乘法封闭，构成一个环结构。

带余除法定理（Division Algorithm）

在 $F[x]$ 中，对于多项式 $f(x)$ 和非零多项式 $g(x)$，存在唯一的多项式 $q(x)$（商）和 $r(x)$（余式），使得：

$$f(x) = q(x)g(x) + r(x)$$

且满足 $r(x) = 0$ 或 $\deg(r) < \deg(g)$。这是多项式欧几里得算法的基础。

3.2 不可约多项式（素多项式）

不可约多项式在多项式环中的地位，等同于素数在整数环中的地位。它们是"不可分割"的基本构建块。

关联知识点：本原多项式的选取标准

定义 3.1：不可约性（Irreducibility）

若次数大于0的多项式 $f(x) \in F[x]$ 不能分解为两个次数更低且系数仍属于 $F$ 的多项式的乘积，则称 $f(x)$ 在域 $F$ 上不可约。否则称为可约（Reducible）。

⚠️ 相对性原则

多项式的可约性完全依赖于底层域 $F$！同一个多项式，在 $\mathbb{Q}$ 上可能不可约，而在 $\mathbb{R}$ 或 $\mathbb{C}$ 上可能可约。例如：$x^2-2$ 在 $\mathbb{Q}$ 上不可约（$\sqrt{2} \notin \mathbb{Q}$），但在 $\mathbb{R}$ 上可约：$x^2-2 = (x-\sqrt{2})(x+\sqrt{2})$。

低次多项式可约性判别法则

对于次数为 $2$ 或 $3$ 的多项式 $f(x) \in F[x]$：

$f(x)$ 在 $F$ 上可约 $\iff$ $f(x)$ 在 $F$ 中有根。

注意：此法则仅适用于2次和3次多项式。对于4次及以上多项式（如 $x^4+x^2+1 = (x^2+x+1)(x^2-x+1)$），无根不代表不可约。

3.3 因式定理与代数基本定理

根的存在与多项式降次分解息息相关。

关联知识点：素数在整数中的类比地位

因式定理（Factor Theorem / 定理2.33）

对于 $f(x) \in F[x]$ 和元素 $\beta \in F$：

$$f(\beta) = 0 \iff (x - \beta) \mid f(x)$$

即 $\beta$ 是多项式方程的根，当且仅当 $(x-\beta)$ 是多项式的因式。证明基于带余除法：$f(x) = (x-\beta)q(x) + r$，其中 $r = f(\beta)$ 是常数。

代数基本定理（Fundamental Theorem of Algebra）

任何 $n \ge 1$ 次的复系数多项式在复数域 $\mathbb{C}$ 中恰好有 $n$ 个根（计重数）。

这意味着在 $\mathbb{C}[x]$ 中，所有的不可约多项式都只能是一次的。在 $\mathbb{R}[x]$ 中，不可约多项式只能是一次或判别式小于0的二次多项式。

3.4 多项式 GCD 与互素

正如整数一样，多项式也有最大公因子和欧几里得算法。这在计算有限域元素的乘法逆元时起决定性作用。

关联知识点：有限域元素的逆元求解

多项式最大公因子

多项式 $f(x)$ 和 $g(x)$ 的最大公因子 $\gcd(f(x), g(x))$ 是能同时整除两者的最高次多项式（通常约定为首一多项式）。若 $\gcd(f,g) = 1$，则称两者互素。

多项式欧几里得算法

基于带余除法：$\gcd(f, g) = \gcd(g, f \bmod g)$。

密码学应用：在有限域 $GF(p^m)$ 中，域元素由多项式 $a(x)$ 表示，模不可约多项式 $P(x)$。由于 $P(x)$ 不可约，只要 $a(x) \neq 0$，必有 $\gcd(a(x), P(x)) = 1$。通过扩展欧几里得算法求解 $a(x)u(x) + P(x)v(x) = 1$，即可得到逆元 $a(x)^{-1} \equiv u(x) \pmod{P(x)}$。

3.1 因式定理验根 基础
判断 $f(x) = x^3 - 4x^2 + x + 6$ 在 $\mathbb{Q}$ 上是否有根，并将其完全因式分解。
考点有理根候选法 + 因式定理。
  1. 备选根：常数项 $6$ 的约数 $\pm 1, \pm 2, \pm 3, \pm 6$。
  2. 验根：$f(-1) = -1 - 4 - 1 + 6 = 0$，故 $x = -1$ 是一个根。
  3. 多项式除法：$(x^3 - 4x^2 + x + 6) \div (x + 1) = x^2 - 5x + 6$。
  4. 继续分解：$x^2 - 5x + 6 = (x - 2)(x - 3)$。
  5. 结论：$f(x) = (x+1)(x-2)(x-3)$，三个根为 $-1, 2, 3$。
✅ 总结：寻找首个有理根是降次分解的突破口，找到一个根 $\beta$ 即可提取因式 $(x-\beta)$，将三次问题转化为二次问题。
3.2 可约性域比较 基础
比较 $x^2 - 2$ 在有理数域 $\mathbb{Q}$、实数域 $\mathbb{R}$、复数域 $\mathbb{C}$ 中的可约性。
考点多项式可约性的相对性原则。
  1. 在 $\mathbb{Q}$ 上：令 $x^2 - 2 = 0$ 得根 $x = \pm \sqrt{2} \notin \mathbb{Q}$。且2次多项式在 $\mathbb{Q}$ 上无根等价于不可约，故在 $\mathbb{Q}$ 上不可约。
  2. 在 $\mathbb{R}$ 上：$\sqrt{2} \in \mathbb{R}$，可分解为 $x^2 - 2 = (x - \sqrt{2})(x + \sqrt{2})$，因式系数属于 $\mathbb{R}$，故在 $\mathbb{R}$ 上可约。
  3. 在 $\mathbb{C}$ 上：因为 $\mathbb{R} \subset \mathbb{C}$，既然在 $\mathbb{R}$ 上可约，在 $\mathbb{C}$ 上必然可约（分解形式同上）。
✅ 总结：这道题深刻体现了"可约性是多项式与底层域的相对属性，不是多项式固有的绝对属性"这一核心理念。
3.3 $GF(2)$ 上的多项式加法 基础
在 $GF(2)[x]$ 中计算 $(x^3 + x + 1) + (x^3 + x^2 + 1)$。
考点$GF(2)$ 中的系数运算本质是模2加法（异或，XOR），相同项抵消。
  1. 合并同类项：$(1+1)x^3 + (0+1)x^2 + (1+0)x + (1+1)$。
  2. 在 $GF(2)$ 中，$1+1=0$。
  3. 化简：$0x^3 + 1x^2 + 1x + 0 = x^2 + x$。
✅ 总结：密码学中广泛使用 $GF(2)$ 上的多项式，其加法和减法完全等效，没有进位/借位，运算速度极快。
3.4 $GF(2)$ 上不可约性判断 中等
判断 $f(x) = x^2 + x + 1$ 在 $GF(2)$ 上是否不可约。
考点$GF(2)$ 上2次多项式不可约 $\iff$ 在 $GF(2)=\{0, 1\}$ 中无根。
  1. 检验 $x=0$：$f(0) = 0^2 + 0 + 1 = 1 \neq 0$。
  2. 检验 $x=1$：$f(1) = 1^2 + 1 + 1 = 1 + 1 + 1$。在 $GF(2)$ 中，$1+1=0$，故 $1+1+1 = 0+1 = 1 \neq 0$。
  3. 结论：由于 $GF(2)$ 中仅有的两个元素都不是 $f(x)$ 的根，且 $f(x)$ 次数为2，故其在 $GF(2)$ 上不可约。
✅ 拓展：$x^2+x+1$ 是构造有限域 $GF(2^2) = GF(4)$ 的不可约多项式。
3.5 有理根定理综合 中等
确定 $f(x) = 2x^3 - 3x^2 - 8x + 12$ 在 $\mathbb{Q}$ 上是否有有理根，并完全分解。
考点有理根定理：根形如 $p/q$，其中 $p$ 整除常数项，$q$ 整除最高次项系数。
  1. 备选根：$p \mid 12 \Rightarrow p \in \{\pm1, \pm2, \pm3, \pm4, \pm6, \pm12\}$；$q \mid 2 \Rightarrow q \in \{1, 2\}$。
    可能的有理根包括整根以及半整根 $\pm 1/2, \pm 3/2$。
  2. 验根：尝试 $x = 3/2$：$f(3/2) = 2(27/8) - 3(9/4) - 8(3/2) + 12 = 27/4 - 27/4 - 12 + 12 = 0$ ✓。
  3. 提取因式：存在因式 $(x - 3/2)$，或者等价的整系数形式 $(2x - 3)$。
  4. 长除法：$(2x^3 - 3x^2 - 8x + 12) \div (2x - 3) = x^2 - 4$。
  5. 最终分解：$x^2 - 4 = (x-2)(x+2)$。
    结论：$f(x) = (2x - 3)(x - 2)(x + 2)$。
✅ 技巧：不要盲目试根，有理根定理将无限的有理数缩小到了有限的候选集。分组分解法也可解此题：$x^2(2x-3) - 4(2x-3) = (x^2-4)(2x-3)$。
3.6 多项式长除法 中等
计算 $f(x) = x^4 + 2x^3 - x - 2$ 除以 $g(x) = x^2 + x - 1$ 的商和余式。
考点多项式带余除法的规范操作：首项相消，逐步降次。
  1. $x^4 \div x^2 = x^2$（第一项商）。减去 $x^2(x^2+x-1) = x^4+x^3-x^2$，余式为 $x^3+x^2-x-2$。
  2. $x^3 \div x^2 = x$（第二项商）。减去 $x(x^2+x-1) = x^3+x^2-x$，余式为 $-2$。
  3. $-2$ 次数为 $0 < \deg(g)=2$，除法终止。
  4. 结论：商 $q(x) = x^2 + x$，余式 $r(x) = -2$。
✅ 验证：$(x^2+x)(x^2+x-1) - 2 = x^4+x^3-x^2+x^3+x^2-x-2 = x^4+2x^3-x-2 = f(x)$ ✓
3.7 $GF(2)$ 上的多项式乘法 中等
在 $GF(2)[x]$ 中，计算 $(x^2 + x + 1) \cdot (x + 1)$。
考点常规展开 + $GF(2)$ 系数模2化简。
  1. 分配律展开：$x(x^2+x+1) + 1(x^2+x+1) = x^3+x^2+x + x^2+x+1$。
  2. 合并同类项：$x^3 + (1+1)x^2 + (1+1)x + 1$。
  3. 系数模2：在 $GF(2)$ 中 $1+1=0$，得 $x^3 + 0x^2 + 0x + 1$。
  4. 结论：结果为 $x^3 + 1$。
✅ 总结：这实际上是代数恒等式 $a^3-b^3 = (a-b)(a^2+ab+b^2)$ 在特征为2的域上的变体。特征2域中 $a^3+b^3 = a^3-b^3$ 且 $(x+1)^3 \neq x^3+1$，但 $(x+1)(x^2+x+1) = x^3+1$ 成立。
3.8 不可约多项式验证 (AES相关) 中等
验证 $p(x) = x^3 + x + 1$ 在 $GF(2)$ 上不可约。
考点3次多项式在 $GF(2)$ 上不可约 $\iff$ 无根。
  1. 若 $p(x)$ 可约，由于它是3次，必定包含一个1次因式，即在 $GF(2)$ 中必定有根。
  2. 验根 $x=0$：$p(0) = 0 + 0 + 1 = 1 \neq 0$。
  3. 验根 $x=1$：$p(1) = 1 + 1 + 1 = 1 \neq 0$（在 $GF(2)$ 下）。
  4. 结论：在 $GF(2)$ 中无根，故不可约。
✅ 拓展：这是 $GF(2^3)$ 构造中的常用模多项式。类似地，AES算法 $GF(2^8)$ 使用的是8次不可约多项式 $x^8+x^4+x^3+x+1$。
3.9 多项式互素验证 中等
验证 $f(x) = x^2 + 1$ 和 $g(x) = x^2 + x + 1$ 在 $GF(2)$ 上互素。
考点使用多项式欧几里得算法计算 $\gcd$。
  1. 在 $GF(2)$ 下计算 $(x^2+x+1) \div (x^2+1)$：
    商1，减去 $1\cdot(x^2+1)=x^2+1$。
    余式为 $(x^2+x+1)-(x^2+1) = x$。
  2. 计算 $(x^2+1) \div x$：
    商 $x$，减去 $x\cdot x = x^2$。
    余式为 $1$。
  3. 计算 $x \div 1$，余式为 $0$。
  4. 结论：最后一个非零余数为 $1$，故 $\gcd(f,g) = 1$，二者互素。
✅ 替代解法：已知 $g(x) = x^2+x+1$ 在 $GF(2)$ 上不可约。任何多项式与不可约多项式要么互素，要么是其倍数。显然 $f(x)$ 并非 $g(x)$ 的倍数，故二者互素。
3.10 重根与重因式判断 综合
设 $f(x) = x^3 - 3x^2 + 4$，验证 $x=2$ 是其二重根并给出完全分解。
考点多重根判定定理：$x=\beta$ 是 $k$ 重根 $\iff$ $f(\beta)=0, f'(\beta)=0, \dots, f^{(k-1)}(\beta)=0$ 且 $f^{(k)}(\beta)\neq 0$。
  1. 验证原函数：$f(2) = 8 - 12 + 4 = 0$ ✓。
  2. 验证一阶导数：$f'(x) = 3x^2 - 6x$，$f'(2) = 12 - 12 = 0$ ✓。
  3. 验证二阶导数：$f''(x) = 6x - 6$，$f''(2) = 12 - 6 = 6 \neq 0$。故 $x=2$ 恰好是二重根。
  4. 长除法分解：已知有因式 $(x-2)^2 = x^2-4x+4$。
    $(x^3 - 3x^2 + 4) \div (x^2 - 4x + 4) = x + 1$。
  5. 结论：$f(x) = (x-2)^2(x+1)$。
✅ 总结：导数法则（形式导数）在代数中判定多重根极其有效，即便在不讨论极限的抽象域中，形式导数法则依然成立。
3.11 系数域扩张效果全面对比 综合
将 $f(x) = x^4 + 1$ 在 $\mathbb{Q}, \mathbb{R}, \mathbb{C}$ 三个域上分别进行因式分解。
考点四次多项式在不同域上的分解形态（域扩张的直观展示）。
  1. 在 $\mathbb{Q}$ 上：无有理根，且不能表示为两个有理系数二次多项式的乘积。故在 $\mathbb{Q}$ 上不可约，$x^4+1$。
  2. 在 $\mathbb{R}$ 上：配方构造平方差。
    $x^4 + 1 = (x^4 + 2x^2 + 1) - 2x^2 = (x^2+1)^2 - (\sqrt{2}x)^2$
    $= (x^2 + \sqrt{2}x + 1)(x^2 - \sqrt{2}x + 1)$。
    由于 $\sqrt{2} \in \mathbb{R}$，此为 $\mathbb{R}$ 上的分解，两个二次因式判别式 $\Delta = 2-4=-2 < 0$，在 $\mathbb{R}$ 上不可再分。
  3. 在 $\mathbb{C}$ 上：由代数基本定理，完全分解为4个一次因式：
    $(x - e^{i\pi/4})(x - e^{3i\pi/4})(x - e^{-i\pi/4})(x - e^{-3i\pi/4})$
    $= (x - \frac{\sqrt{2}+i\sqrt{2}}{2})(x - \frac{-\sqrt{2}+i\sqrt{2}}{2})(x - \frac{\sqrt{2}-i\sqrt{2}}{2})(x - \frac{-\sqrt{2}-i\sqrt{2}}{2})$
✅ 总结："数域越大，拆解越碎"。这正反映了为什么我们需要对有限域进行扩张——为了能构造出更多的运算元素和性质。
3.12 $GF(2)$ 模多项式乘法运算 综合
在有限域 $GF(2^3) \cong GF(2)[x]/(x^3+x+1)$ 中，计算 $x^2 \cdot (x+1)$。
考点有限域乘法的两阶段操作：先普通多项式乘法，再按不可约多项式取模（降次）。
  1. 代数乘法：$x^2 \cdot (x+1) = x^3 + x^2$。
  2. 取模降次：模多项式为 $x^3+x+1 = 0 \pmod{x^3+x+1}$。
    在 $GF(2)$ 中，移项等价于相加：$x^3 \equiv x + 1$。
  3. 替换高次项：将 $x^3$ 替换为 $x+1$。
    $x^3 + x^2 \equiv (x+1) + x^2 = x^2 + x + 1$。
  4. 结论：运算结果为 $x^2 + x + 1$。
✅ 总结：这种"模不可约多项式"的运算结构是整个AES算法列混淆操作的底层数学原理。$x^n \equiv \dots$ 降次法则极其实用。
3.13 不可约多项式枚举 综合
$GF(2)$ 上有多少个次数为 $2$ 的不可约多项式？列举出它们。
考点通过排除法找出所有无根的低次多项式。
  1. 所有2次多项式：形式为 $ax^2+bx+c$，由于次数为2，必有 $a=1$。共有 $2^2 = 4$ 个首一多项式：
    $x^2, x^2+1, x^2+x, x^2+x+1$
  2. 排除可约多项式（在 $GF(2)$ 中有根的）：
    $x^2$：根为0，可约（$x \cdot x$）
    $x^2+x$：根为0和1，可约（$x(x+1)$）
    $x^2+1$：根为1，可约（因为 $(x+1)^2 = x^2+2x+1 \equiv x^2+1 \pmod 2$）
  3. 寻找不可约：剩下 $x^2+x+1$。验根：$f(0)=1, f(1)=3 \equiv 1$，均非零。
  4. 结论：只有 $1$ 个：$x^2+x+1$。
✅ 总结：这是构建有限域 $GF(4)$ 的唯一多项式选项。$GF(2)$ 上的 $(x+y)^2 = x^2+y^2$（"大一新生的梦想"等式）是由于特征为2消去了交叉项 $2xy$。
3.14 多项式 GCD 综合求解 综合
在 $GF(2)[x]$ 中，求 $\gcd(x^4+x^3+x^2+x, x^3+x^2+x+1)$。
考点多项式欧几里得除法 / 提取公因式法。
  1. 解法一：欧几里得法
    $(x^4+x^3+x^2+x) \div (x^3+x^2+x+1)$：
    商为 $x$：$x(x^3+x^2+x+1) = x^4+x^3+x^2+x$。
    两式相减，余数为 $0$！
    这意味着第二个多项式直接整除第一个。
  2. 解法二：因式分解法
    $f_1 = x(x^3+x^2+x+1)$
    $f_2 = x^3+x^2+x+1$
    显然 $\gcd = f_2$。
  3. 结论：$\gcd$ 为 $x^3+x^2+x+1$。
✅ 总结：做除法前先观察表达式结构。此题由于项数的巧合，提取一个 $x$ 即可看出倍数关系，极大简化了计算。
3.15 分裂域概念初步 综合
$f(x) = x^2 - 2$ 在 $\mathbb{Q}$ 上的分裂域（Splitting Field）是什么？它的扩域维度是多少？
考点分裂域定义：使多项式能完全分解为一次因式的"最小"扩域。
  1. 求解根：$x^2-2 = 0 \Rightarrow x = \pm\sqrt{2}$。
  2. 构造扩域：为包含所有根，需要将 $\sqrt{2}$ 添加到基础域 $\mathbb{Q}$ 中，得到 $\mathbb{Q}(\sqrt{2}) = \{a + b\sqrt{2} \mid a,b \in \mathbb{Q}\}$。
  3. 验证分裂：在 $\mathbb{Q}(\sqrt{2})$ 中，$x^2-2 = (x-\sqrt{2})(x+\sqrt{2})$ 完全分解。
  4. 计算维度：集合 $\{1, \sqrt{2}\}$ 构成了 $\mathbb{Q}(\sqrt{2})$ 作为 $\mathbb{Q}$-线性空间的一组基，因此扩域维度 $[\mathbb{Q}(\sqrt{2}):\mathbb{Q}] = 2$。
✅ 总结：分裂域概念正是有限域 $GF(p^m)$ 构造的核心原型。$GF(p^m)$ 实际上就是 $GF(p)$ 上多项式 $x^{p^m}-x$ 的分裂域。
